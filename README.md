# 스템케어재팬 (StemCareJapan) - 홈페이지

## 📋 프로젝트 개요

**한국 환자를 위한 일본 줄기세포 치료 의료관광 토탈케어 서비스** 홈페이지입니다.

자택 픽업부터 일본 귀환까지 줄기세포 치료, 통역, 호텔, 관광 등 모든 일정을 관리하는 토탈케어 서비스를 소개하는 반응형 웹사이트입니다.

---

## ✅ 구현된 기능

### 페이지 구성 섹션
- **히어로(Hero)**: 서비스 핵심 메시지, 주요 통계(누적 환자 500+, 만족도 98%, 제휴병원 5+, 24h 케어)
- **소개(About)**: 일본 줄기세포 치료 필요성 및 스템케어재팬 소개
- **서비스(Services)**: 6대 토탈케어 서비스 상세 소개 (국내 에스코트, 일본 공항, 숙박, 치료, 관광, 24h 케어)
- **이용 방법(Process)**: 6단계 프로세스 타임라인 (상담→플랜→준비→출발→치료/관광→귀국 후 관리)
- **치료 안내(Treatment)**: 치료 유형(자가/동종/엑소좀), 치료 가능 질환 6분야, 제휴 병원 3곳
- **관광 안내(Tourism)**: 도쿄/오사카/후쿠오카/교토 탭별 상세 안내 + 3종 패키지
- **고객 후기(Testimonials)**: 슬라이더 형식 4개 실 후기
- **FAQ**: 치료 관련 4문항 + 서비스 관련 4문항 아코디언
- **문의(Contact)**: 무료 상담 신청 폼 (이름/연령/연락처/이메일/질환/패키지/내용)
- **푸터(Footer)**: 연락처, 링크, SNS, 법적 고지

## 🌐 다국어 지원 (i18n)

네비게이션 우측의 **KO | JA | EN** 버튼으로 언어 전환이 가능합니다.

| 언어 | 적용 범위 |
|------|-----------|
| 한국어 (KO) | 전체 기본 언어 |
| 일본어 (JA) | 네비·히어로·도시 탭 콘텐츠 |
| 영어 (EN) | 네비·히어로·도시 탭 콘텐츠 |

- 선택 언어는 `localStorage`에 저장되어 새로고침 후에도 유지됩니다.
- 번역 데이터는 `js/main.js`의 `i18nData` 객체에 저장됩니다.
- HTML 요소에 `data-i18n="key"` 또는 `data-i18n-html="key"` 속성으로 번역 키를 지정합니다.
- 추가 번역이 필요한 경우 `i18nData` 객체에 키를 추가하고, 해당 HTML 요소에 `data-i18n` 속성을 부여하면 됩니다.

## 🖼️ 이미지 적용 방법

1. [Unsplash](https://unsplash.com) 또는 [Pexels](https://pexels.com)에서 무료 이미지 다운로드
2. `images/` 하위 해당 폴더에 저장
3. `index.html`의 placeholder div를 `<img>` 태그로 교체:

```html
<!-- 변경 전 -->
<div class="city-img-placeholder city-img-tokyo">
  <i class="fas fa-torii-gate"></i>
  <p>도쿄</p>
</div>

<!-- 변경 후 -->
<img src="images/tourism/tokyo/tokyo-main.jpg"
     alt="도쿄 관광" class="city-img" loading="lazy" />
```

> 상세 가이드: `images/README.md` 참고

## 🗂️ 데이터 모델

### `inquiries` 테이블 (RESTful API)
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 ID (자동) |
| name | text | 신청자 이름 |
| age | number | 연령 |
| phone | text | 연락처 |
| email | text | 이메일 |
| disease | text | 주요 증상/질환 |
| package | text | 관심 패키지 |
| message | rich_text | 문의 내용 |
| submitted_at | datetime | 신청 일시 |

**API 엔드포인트**: `GET/POST tables/inquiries`

> 현재 백엔드 미연동 — 폼 제출 시 성공 모달만 표시. 백엔드 연동은 추후 개발 예정.

## 🚧 추후 개발 권장 사항

8. **예약 시스템** 연동 (캘린더 기반 일정 예약)
